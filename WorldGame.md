# 材质函数

## WorldAlignedBlend

​	通过使用World Aligned Blend节点，我们能够拾取到景观的斜面遮罩，并还能自由控制偏移位置。

PARAM：

​	In Explicit Normal (V3):

​		Explicit Normal 的翻译是“显式法线”。当混合法线贴图时需要保留噪声时，这个参数非常有用。

​	In World Vector (V3):

​	Blend Sharpness (S):

​		控制混合过渡值

​	Blend Bias (S):

​		控制混合高度偏移位置

​	Clamped? (B):

​	Alpha (S):

OUTPUT：

​	Alpha:

​	w/Vertex Normals:

​	W/ Explicit Normal:
